# Welcome to the Crossing Beams and Crystals Under Pressure repository

Welcome to the **Crossing Beams and Crystals Under Pressure** repository.
 This repository contains the graduation work of a **Game Development student (HBO-ICT, Game Development)**, created in collaboration with the **Smart Education Lab**.

Both projects were created using **Processing 3**.

------

## **Games Included**

This repository contains **two separate games**:

- **Crossing Beams**
- **Crystals Under Pressure**

Each game has:

- Its own **source code**, located in the respective subfolder.
- A **Windows executable build**, available via the `.zip` files in the repository.

> ⚠️ **Note:**
>  The provided builds currently only include **Windows executables**. To run the games on macOS or Linux, please use the Processing 3 source code and export the game on your platform.

------

## How to Play**

### **Option 1 — Download Executables**

1. Download the `.zip` of each game from the repository.
2. Extract the folder.
3. Run the `.exe` file inside.

### **Option 2 — Run from Source (Processing 3)**

1. Install **Processing 3** (https://processing.org/releases/).
2. Open the game folder in Processing.
3. Run the project via the **Play** button.
